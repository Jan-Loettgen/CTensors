Applies input function to all elements, and then returns what the function returns on the last element. 

Note this function is different from function \textquotesingle{}mat\+\_\+apply\+\_\+func\textquotesingle{} because that function returns a matrix that has had the function applied to it. This function instead applies the input function to all elements and returns the function return, on the final element. This method could be used to calculate the sum of all elements of a matrix by using static variables in the input function.

To accomadate static variables the function should take as input a reset value. This will be 1 on the last element of the matrix, and will be 0 otherwise.
